# Claude Code — Cheat Sheet

Une référence interactive complète pour [Claude Code](https://claude.ai/code), l'outil CLI d'Anthropic.

## Aperçu

Interface web minimaliste regroupant toutes les commandes, raccourcis et configurations utiles pour travailler efficacement avec Claude Code au quotidien.

## Contenu

| Onglet | Description |
|--------|-------------|
| **CLI** | Installation, modes d'exécution, flags principaux et avancés |
| **Slash Commands** | Commandes `/`, mémoire, hooks, MCP, plugins |
| **Raccourcis Clavier** | Contrôles, modes, saisie multiligne, préfixes |
| **Mode Vim** | Navigation, édition, changement de mode |
| **Configuration** | CLAUDE.md, settings.json, permissions, hooks |
| **Avancé** | Pipes, sous-agents, MCP, git worktrees, CI/CD, plugins |

## Fonctionnalités

- **Recherche globale** — `Ctrl+K` pour filtrer toutes les commandes en temps réel
- **Copie au clic** — cliquer sur n'importe quelle ligne copie la commande dans le presse-papiers
- **Navigation par onglets** — accès rapide à chaque catégorie
- **Design responsive** — utilisable sur mobile

## Structure

```
claude-code-cheatsheet/
├── index.html    # Structure HTML
├── styles.css    # Styles et thème sombre
└── script.js     # Données et logique interactive
```

## Usage

Ouvrir `index.html` directement dans un navigateur — aucune dépendance, aucun build requis.
